#### OCR (Optical Character Recogination)
* Tesseract is one of the popular OCR Engine for various OS.
* Alternative to Tesseract are :
    * Tensorflow
    * OpenCV
    * Google Cloud Vision API
    * Amazon Rekognition
    * Tesseract
    * EasyOCR
    * scanR
    * ZXing
* Pytesseract accept `RGB` values.
